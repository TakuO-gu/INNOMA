# 2026-02-25 更新履歴

## 新機能追加

- INNOMAデモアプリを `/demo` ルートに新設
  - **トップページ** (`/demo`): 2つの体験への入り口。ポップなデザイン
  - **A. 情報探しタイムトライアル** (`/demo/time-trial`): スロットマシンで自治体×情報カテゴリを選び、既存サイトとINNOMAサイトで同じ情報を探す時間を比較
  - **B. イノマ工場見学** (`/demo/factory`): 5ステップの体験型UIで、既存サイト→テンプレートクローン→AI情報収集→ドラフト承認→完成ページの流れを体験
- クイズデータ自動生成ロジック (`lib/demo/quiz-data.ts`): デプロイ済み25自治体のvariablesから手数料・電話番号等のクイズを自動生成
- クイズデータAPI (`/api/demo/quiz`): クイズデータをJSON形式で提供
