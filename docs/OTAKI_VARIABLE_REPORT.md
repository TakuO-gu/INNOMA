## 4. 備考

### 4.1 王滝村の特徴

- **人口規模**: 約660人の小規模自治体
- **地理**: 御嶽山の麓、木曽郡の最西端
- **特徴的な支援**:
  - 保育料完全無料
  - 学校給食費無料
  - 18歳まで医療費無料
  - 誕生支援金・進級支援金制度

### 4.2 公式サイトの技術的特徴

- JavaScriptによる動的コンテンツ読み込みが多い
- 一部ページは静的HTML、一部はテンプレートから動的生成
- PDFでの情報提供が多い（広報誌、回覧文書、各種計画等）

### 4.3 調査時の注意点

- 小規模自治体のため、複数の業務を1つの課で担当
- 公式サイトのURL構造が一部変更されている可能性（httpとhttpsの混在、旧URLの404エラー）
- 最新情報は広報誌や回覧文書で周知されることが多い
