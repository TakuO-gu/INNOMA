## コンポーネント一覧

### 1. コンテンツ構造化コンポーネント

#### Accordion（アコーディオン）
- **用途**: 展開/折畳みで情報を整理
- **適用ケース**: FAQ、詳細情報の段階的表示、長いコンテンツの整理
- **構成**: Accordion > AccordionSummary + AccordionContent
- **ガイドライン**: 準備中

#### Disclosure（ディスクロージャー）
- **用途**: 単一の展開/折畳み要素
- **適用ケース**: 補足情報、「詳しく見る」的な追加情報
- **ガイドライン**: 準備中

#### Table（テーブル/データテーブル）
- **用途**: データや情報を行と列で構造化
- **適用ケース**:
  - 複数データの比較
  - 構造化されたデータ表示
  - 料金表、一覧表
- **使用すべきでない場合**:
  - 単純な名前-値ペア → 説明リスト（Dl）を使用
  - レイアウト目的

#### Dl（説明リスト）
- **用途**: 用語とその説明、名前-値ペアの表示
- **適用ケース**:
  - 申請に必要な書類一覧
  - 問い合わせ先情報
  - 手続きの詳細情報
- **ガイドライン**: 準備中

#### List（リスト: Ol/Ul）
- **用途**: 順序付き/順序なしリスト
- **適用ケース**:
  - Ol: 手順、優先順位のある項目
  - Ul: 持ち物、条件、注意事項
- **ガイドライン**: 準備中

#### Blockquote（引用ブロック）
- **用途**: 引用文や強調テキストの表示
- **適用ケース**: 法令の引用、重要な規則
- **ガイドライン**: 準備中

#### Divider（ディバイダー）
- **用途**: セクション間の視覚的区切り
- **適用ケース**:
  - 異なるセクションの区切り
  - リスト内のグループ分け
- **使用ガイド**:
  - リスト内: 最低8pxの余白
  - セクション間: 最低16pxの余白
  - ディバイダーなしでも意味が通じる構造が前提

### 2. 通知・注意喚起コンポーネント

#### NotificationBanner（ノティフィケーションバナー）
- **用途**: ユーザーに重要情報を通知
- **タイプ**:
  | タイプ | 用途 |
  |--------|------|
  | success | 完了通知 |
  | error | エラー・失敗 |
  | warning | 警告・注意 |
  | info-1 | 情報（プライマリカラー） |
  | info-2 | 情報（ニュートラル） |
- **スタイル**:
  - Standard: 丸みを帯びたボーダー
  - Color Chip: 左側に太いアクセントボーダー
- **必須パーツ**: アイコン、タイトル
- **オプション**: 閉じるボタン、日時、説明文、アクションボタン
- **配置**: 通常はページ上部（最初のビューポート内）

#### EmergencyBanner（緊急時バナー）
- **用途**: 最優先の緊急情報告知
- **特徴**: オレンジ色の6px枠線、白背景
- **適用ケース**: 災害情報、緊急のお知らせ

### 3. ナビゲーションコンポーネント

#### Breadcrumbs（パンくずリスト）

- **ガイドライン**: 準備中

#### ProgressIndicator（プログレスインジケーター）
- **用途**: 進行状況の表示
- **ガイドライン**: 準備中

### 4. カード・リストコンポーネント

#### Card（カード）
- **用途**: 単一トピックに関連するコンテンツをグループ化
- **適用ケース**:
  - メディアとテキストの組み合わせ
  - 類似アイテムのリスト/グリッド表示
  - 記事、サービス一覧
- **使用すべきでない場合**:
  - データ比較・分析 → テーブルを使用

#### ResourceList（リソースリスト）
- **用途**: 関連リソースの一覧表示
- **適用ケース**: 関連リンク、参考資料
- **ガイドライン**: 準備中

---

### 5. ボタン・リンクコンポーネント

#### Button（ボタン）
- **用途**: アクション実行またはページ遷移のトリガー
- **バリエーション**:
  | スタイル | 重要度 | 用途 |
  |---------|--------|------|
  | Filled | 高 | 主要アクション（1画面に1つまで） |
  | Outline | 中 | 副次アクション（同一コンテキスト内3つまで） |
  | Text | 低 | キャンセル等 |
- **サイズ**: Large(136×56), Medium(96×48), Small(80×36), X-Small(72×28)
- **使用すべきでない場合**:
  - disabled属性での非活性化は避ける
  - 代わりに必要なアクションを明示

#### UtilityLink（ユーティリティリンク）
- **用途**: 補助的なリンク
- **ガイドライン**: 準備中

## INNOMAサービスページでの推奨使用パターン

### 情報タイプ別コンポーネント選択

| 情報タイプ | 推奨コンポーネント |
|-----------|-------------------|
| 手続きの流れ | StepNavigation, Ol |
| 必要書類 | Dl, Ul |
| 問い合わせ先 | Dl, Card |
| 注意事項 | NotificationBanner (warning), Ul |
| 緊急情報 | EmergencyBanner |
| FAQ | Accordion |
| 料金・費用 | Table |
| 対象条件 | Ul, Dl |
| 関連サービス | Card, ResourceList |
| 法令引用 | Blockquote |
| カテゴリ表示 | ChipTag |
| ステータス | StatusBadge |
