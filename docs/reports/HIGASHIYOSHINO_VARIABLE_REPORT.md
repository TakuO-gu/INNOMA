## 4. 備考

### 4.1 公式サイトについて

- SSL証明書に問題があり（ERR_TLS_CERT_ALTNAME_INVALID）、HTTPSでのアクセスが困難
- HTTPでアクセス可能: http://www.vill.higashiyoshino.nara.jp/
- 情報が複数ページに分散しており、体系的な一覧ページが少ない

### 4.2 小規模自治体の特徴

- 人口約1,500人の小規模自治体
- 総務企画課、住民福祉課、税務保険課、産業建設課の4課体制
- 代表電話番号（0746-42-0441）で各課に取り次ぎ
- 吉野広域行政組合（吉野町・川上村・東吉野村）でごみ処理等を共同実施

### 4.3 子育て支援の充実

- 子どもの医療費完全無料化
- 預かり保育、学童保育の実施
- 東吉野こども園（幼稚園＋小規模保育事業所）の整備

### 4.4 移住促進

- 空き家バンク制度あり
- 移住相談窓口: 東吉野村 総務企画課
- 薪ストーブ購入補助（上限15万円）
- 太陽光発電システム設置費補助
