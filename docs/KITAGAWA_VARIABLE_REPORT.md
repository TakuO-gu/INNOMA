## 概要

北川村は高知県東部の安芸郡に属する村。「ゆず王国」として知られ、日本有数のゆずの産地。幕末の志士・中岡慎太郎の出身地でもある。

| 項目 | 値 | 出典 |
|------|-----|------|
| 人口 | 1,170人（2025年9月30日時点） | 公式サイト |
| 世帯数 | 626世帯 | 公式サイト |
| 男性 | 584人 | 公式サイト |
| 女性 | 586人 | 公式サイト |
| 面積 | 196.73km² | 公式サイト |
| 市町村制施行 | 1889年（明治22年）4月1日 | Wikipedia |

**歴史**:
- 明治22年（1889年）4月1日: 北川村、野友村、加茂村、野川村が合併して安芸郡北川村発足
- 中岡慎太郎館、モネの庭マルモッタン、北川村温泉などの観光施設あり

## 4. 備考

### 4.1 北川村の特徴

- **ゆず王国**: 日本有数のゆずの産地として知られ、村の主要産業
- **中岡慎太郎**: 幕末の志士・中岡慎太郎の出身地（中岡慎太郎館あり）
- **モネの庭**: クロード・モネの自宅の庭を再現した「モネの庭マルモッタン」
- **北川村温泉**: 四国有数の泉質を誇る温泉
- **小規模自治体**: 人口約1,170人、4課体制の役場組織

### 4.2 組織構成

北川村役場は4課＋教育委員会＋議会事務局の構成：
1. 総務課
2. 住民課
3. 産業政策課
4. 経済建設課
5. 教育委員会
6. 議会事務局

### 4.3 医療体制

- 村内に総合病院なし
- 近隣の中核病院は高知県立あき総合病院（安芸市）
- 村内診療所の有無は要確認

### 4.4 アクセス

- 高知市から車で約1時間25分（57.3km）
- 土佐くろしお鉄道ごめんなはり線「奈半利駅」からタクシー約30分
- 奈半利駅からは公共交通機関なし
